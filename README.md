# BanglaHateXAI 🔍

A Transformer-CNN-Attention Framework for Interpretable 
Hate Speech Detection in the Bengali Language

## 📌 Overview
Undergraduate research at Z.H Sikder University of Science 
& Technology (ZHSUST), 2026.
Detects hate speech in Bengali text using Explainable AI (XAI).

## 🛠️ Technologies Used
- Python
- TensorFlow / PyTorch
- Transformer Architecture
- CNN + Attention Mechanism
- Explainable AI (XAI)
- NumPy, Pandas

## 📊 Research Highlights
- Developed interpretable deep learning model for Bengali NLP
- Combined Transformer + CNN + Attention for better accuracy
- Applied XAI techniques for model interpretability
